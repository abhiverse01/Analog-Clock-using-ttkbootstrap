# Modern Analog Clock
This application creates a modern Analog clock using the Tkinter library in Python. It displays the current time with hour, minute, and second hands on a graphical user interface.
Prerequisites

Make sure you have the following installed on your system:
Python 3. x
tkinter package
ttkbootstrap package
ttkthemes package

Installation
Clone or download the source code for the application.
Install the required packages by running the following command:

pip install tkinter ttkbootstrap ttkthemes

Usage
To run the application, execute the following command in the terminal:
python <path_to_file>/modern_analog_clock.py
The application window will appear with a modern Analog clock displaying the current time.
Time Format
By default, the clock is set to the twelve-hour format. You can switch between twelve-hour and twenty-four-hour formats by clicking the "Switch Time Format" button.

Customization
You can customize the application according to your preferences by modifying the code. Some possible customizations include:
Changing the theme of the clock by modifying the theme parameter in the ThemedTk and Style objects.
Modifying the font, size, and colour of the clock's title, numbers, and hands by changing the corresponding parameters in the code.
Adjusting the size and position of the clock canvas and other GUI elements by modifying the relevant parameters.
